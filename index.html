Resuelve el Problema N-Reinas

¿Qué es el problema de las N-Reinas?

El problema N-Reinas consiste en colocar N reinas en un tablero de ajedrez de N x N de tal manera que ninguna reina pueda atacar a otra. Esto significa que no puede haber dos reinas en la misma fila, columna o diagonal.

Pasos para resolverlo:

  1. Seleccionar un número de reinas (N).
  2. Comenzar desde la primera fila y colocar una reina en una posición válida.
  3. Pasar a la siguiente fila, colocando otra reina en una posición válida que no ataque a las anteriores.
  4. Repetir hasta completar todas las filas.
  5. Si no se encuentra una solución, retroceder (backtracking) y probar otra configuración.
